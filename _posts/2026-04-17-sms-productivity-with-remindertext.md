Most people do not need a more complicated planning system. They need a reminder system that shows up at the right moment.

Here are five habits that work well for busy schedules.

## 1. Set reminders at the decision point

If a task has to happen after a trigger (for example, “text Alex after the meeting”), set the reminder immediately while the trigger is fresh in your mind.

## 2. Use short, action-first wording

Keep each reminder direct. Start with the action, then add details only if needed.

Examples:
- "Pay electric bill"
- "Take evening medication"
- "Send project update to team"

## 3. Turn repeated tasks into recurring reminders

Any task you do every week or month should become a recurring reminder once, instead of being re-entered each time.

## 4. Create reminders for transitions

Many missed tasks happen between activities. Use reminders around transition moments like "before leaving work" or "after dinner".

## 5. Review failed or unsent reminders weekly

A quick weekly review helps you catch patterns and improve timing. Small timing tweaks can dramatically improve follow-through.
